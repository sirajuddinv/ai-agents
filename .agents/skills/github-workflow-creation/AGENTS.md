# GitHub Workflow Creation

Companion skill for creating CI/CD workflows following industrial standards.

## Quick Reference

- **Skill SSOT:** [SKILL.md](./SKILL.md)
- **Scripts:** [scripts/](./scripts/)

## When to Apply

Use this skill when:
- Creating new GitHub Actions workflows
- Setting up deployment pipelines
- Configuring access control for workflows
- Optimizing checkout/build/deploy steps

## Key Standards

- Workflows at root `.github/workflows/` (not subdirectories)
- All scripts in `.github/workflows/scripts/`
- Title Case for step names
- Full version pinning for actions
- No inline scripts in YAML
