# Project Structure & Documentation (Ref)

This bridge provides passive context for the `project-structure` skill, which encodes industrial-standard project
folder structure, root hygiene, README conventions, and documentation placement rules — including the canonical
`AGENTS.md` skills table format used at every workspace root.

It should be invoked whenever the user asks to organize a project, fix folder structure, create or audit a
`README.md` / `AGENTS.md`, or normalize the layout of a new repository.

- **Primary Entry Point**: [.agents/skills/project-structure/SKILL.md](./SKILL.md)
